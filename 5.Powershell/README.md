# References

** Politique d'execution

- Etablir


```
PS > Set-ExecutionPolicy RemoteSigned
```

Avec Scope


```
PS > Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine
```

- Enlever

```
PS > Set-ExecutionPolicy Restricted
```
